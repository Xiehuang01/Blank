Blank 后端代码生成 — 实现计划
概述
Blank 是一个明信片社交应用（Vue 3 前端），当前所有数据存储在 localStorage。本计划将为其生成完整的 Node.js 后端，使用 Express + MySQL 8 + Redis + JWT + Nodemailer 技术栈。

前端分析总结
通过分析15个Vue页面和2个Store文件，归纳出以下功能模块：

模块	页面	核心操作
认证	Login.vue, Register.vue	登录、注册、邮箱验证码
首页广场	Home.vue	发现明信片列表、漂流传递列表、点赞
明信片详情	PostDetail.vue	查看、点赞、评论（置顶/删除）、漂流接力
创建明信片	Create.vue	创建普通/漂流明信片、上传图片、选邮票
邮件箱	Mail.vue	我的明信片列表、翻转查看、批量删除
发件箱	Outbox.vue	已发送明信片列表
商店	Shop.vue	浏览邮票、购买邮票
我的邮票	MyStamps.vue	已购买邮票列表
收藏	Favorites.vue	已点赞的明信片

User Review Required
IMPORTANT

Redis 用途：计划将 Redis 用于：邮箱验证码缓存（TTL 3600s）、JWT Token 黑名单（登出后失效）、签到状态缓存。请确认是否满足需求。

IMPORTANT

DashScope API：env 中配置了 DASHSCOPE_API_KEY，前端 Create.vue 有 AI 智能体按钮但未实现具体调用逻辑。本次后端会预留 AI 接口路由但不实现具体业务逻辑，后续可扩展。

WARNING

图片存储：前端目前使用 base64/外部URL。后端将实现本地文件上传存储到 uploads/ 目录，返回可访问的 URL。生产环境建议改用 OSS。
